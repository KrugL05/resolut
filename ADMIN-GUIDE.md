# Инструкция: как управлять галереей на сайте RESOLUTE

Фотографиями в галерее вы управляете сами через панель администратора —
программист для этого не нужен.

## Вход в панель

1. Откройте в браузере: **`https://ВАШ-САЙТ/_/`**
   (например, `https://resolute-club.ru/_/`)
2. Введите логин (email) и пароль, которые вам выдали.

## Добавить фото в галерею

1. В левом меню выберите коллекцию **`gallery`**.
2. Нажмите **+ New record** (Новая запись).
3. В поле **image** нажмите и выберите фото с компьютера
   (JPG, PNG или WebP, до 10 МБ).
4. *(необязательно)* **sort** — число для порядка показа: чем меньше число,
   тем раньше фото в галерее (например, 10, 20, 30…).
5. *(необязательно)* **caption** — подпись к фото.
6. Нажмите **Create** (Создать).

Фото сразу появится в галерее на сайте — обновите страницу сайта и проверьте.

## Удалить фото

1. Коллекция **`gallery`** → найдите нужную запись.
2. Откройте её и нажмите **Delete** (Удалить), либо отметьте галочкой и
   удалите из списка.

## Поменять порядок фотографий

Откройте запись и измените число в поле **sort**. Меньше число → выше в галерее.
Удобно задавать с шагом 10 (10, 20, 30…), чтобы потом легко вставлять новые
между ними (например, 15).

# Статьи / новости

Блок «Новости клуба» сразу под первым экраном — это карточки, которые
листаются вбок; по клику статья открывается целиком. Управляются они так же
просто, через ту же панель.

## Добавить статью

1. В левом меню выберите коллекцию **`articles`**.
2. Нажмите **+ New record**.
3. Заполните поля:
   - **image** — обложка статьи (фото для карточки и шапки статьи).
   - **title** — заголовок (виден на карточке и вверху статьи).
   - **excerpt** — краткое описание (короткий текст на карточке).
   - **body** — полный текст статьи. Здесь есть редактор с форматированием:
     жирный, списки, ссылки, вставка картинок.
   - **sort** — порядок (меньше число → левее в ленте).
4. Нажмите **Create**.

Статья сразу появится в блоке «Новости» на сайте.

## Редактировать / удалить статью

Коллекция **`articles`** → откройте нужную запись, измените поля и нажмите
**Save**, либо нажмите **Delete**, чтобы удалить.

## Подсказки по статьям

- **excerpt** держите коротким (1–2 строки) — он для карточки-превью.
- Весь длинный текст — в поле **body**, его видно только в раскрытой статье.
- Обложку (**image**) лучше брать горизонтальную — она обрезается под карточку.

---

## Общие подсказки

- Фото лучше загружать **горизонтальные/квадратные**, нормального качества —
  сайт сам уменьшит их для превью.
- Менять что-либо может только тот, у кого есть логин и пароль от панели —
  обычным посетителям сайта она недоступна.
- Если забыли пароль — обратитесь к программисту, он сбросит его на сервере.
